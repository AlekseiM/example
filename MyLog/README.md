# example
Написать собственную систему логирования.
Требуется предусмотреть возможность конфигурирования:

    набора событий для логирования,
    ротации логов,
    использования различных аппендеров (файл, консоль),
    формата вывода данных.


В качестве примера интерфейса разрабатываемой системы предлагается рассматривать slf4j.
При выполнении задания не должны быть использованы существующие библиотеки логирования.

При рассмотрении задания будет приветствоваться, если исходный код будет содержать комментарии.
